Keyboard Shortcuts?

You've got command.

Action Shortcut
Open/Close Settings: Command Comma
Open/Close History: Command H
Share Sheet: Command Shift S
Reload page: Command R
Back: Command Left Arrow
Forward: Command Right Arrow
Return to search engine: Command S
Jump to web address bar Double Click Space


To enable multiple tabs in Xplore, open Xplore and select View in the macOS menu bar, then Show Tab Bar.
